# Projeto Web - Agência de Viagens

Nome: Leonardo  
Matrícula: 907515 

## Proposta do Projeto

O projeto consiste na criação de um site de uma agência de viagens fictícia chamada "Explore Mundo".

O objetivo é apresentar destinos turísticos com imagens e descrições, além de permitir que o usuário se cadastre para receber promoções através de um formulário.

## Funcionalidades

- Menu de navegação
- Exibição de destinos turísticos
- Imagens ilustrativas
- Formulário de cadastro
